In order to use this plugin you may follow the following instractions:
1- Download the .zip "master" version from GitHub:
	https://github.com/ggarreton/GiiAttendance
2- Move the .zip to the directory localhost/moodle/mod/
	*localhost could be the directory of MAMP or XAMPP.
3- Unzip the document.
4- Change the name of the floder created by the unzip (GiiAttendance-master). The new name MUST be: "attendance"
5- Go to localhost/moodle and access with an Administrator account.
6- Access to http://localhost/moodle/admin/index.php and click on "Reload".
7- Click on the botton at the end of the page "Upgrade Moodle database now".
8- Now you are ready to use the plugin, in any course you could "Turn edition on" and "Add an activity or resource". There should be the new plugin, "attendance".

Thanks for install the plugin.
